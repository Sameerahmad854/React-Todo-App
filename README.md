# Todo Manager App

A modern **Todo App** built with **React** using **Create React App**.  
Easily manage your daily tasks with features like adding, editing, deleting, and filtering tasks.

## Features

1.Add, edit, and delete tasks  
2. Mark tasks as completed  
3. Organize tasks with categories/tags  
4. Set priority levels (High / Medium / Low)  
5. Search and filter tasks  
6. Responsive design using **Bootstrap 5**  
7. Persistent storage with browser localStorage

## Getting Started

Follow these steps to run the project locally.

### Installation

1. Clone the repository:
   git clone https://github.com/Sameerahmad854/React-Todo-App.git
   cd React-Todo-App

## Install dependencies:

npm install

## Start the development server:

npm star

Open http://localhost:3000
in your browser.

## Deployment

Build the app for production:
npm run build

## Deploy to GitHub Pages:

npm run deploy
Live demo: https://sameerahmad854.github.io/React-Todo-App/
